As a recent university graduate I don't have much professional work to show yet, but here is some work I've done in my free time.

Projects

Peppermint: Refreshing HTML templating with JavaScript. Very easy to learn, and very powerful. Inspired by JSX and EJS. Published on npm (in alpha).

gulp-peppermint: Gulp wrapper for Peppermint templating engine. Published on npm.

Google Maps Place crawler: Node.js script that uses the Google Places API to programmatically retrieve and compile a list of places (e.g. businesses) in a given area. Data can include place/company name, address, website, phone number, and latitude/longitude.

williams.blue: This website. Built as a single-page application, it is simple by design and easy to use and navigate. It is entirely composed of my own HTML, CSS, and vanilla JavaScript. Take a look at this post for my thoughts on its design and development. View the source on GitHub.
